# git.0.1

------- create Repo --
git init [name]
git clone [URL]


------- Make changes --
git add [file]
git commit -m [commit message]
git reset [file]


------- Synchronise --
git fetch 
git pull
git push


------- observe & monitor --
git status
git diff
git blame [file]
git show [commit]:[file]
git log

------- working with branches --
git branch
git checkout
git merge
git tag [name]

------- command --
git command
